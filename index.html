Project Overview

This tool analyzes seasonal influenza-associated hospitalization data sourced from the FluSurv-NET FluView system (2009–2025). The data reflect confirmed influenza-related hospitalizations across multiple U.S. sites.

By integrating surveillance data with machine learning and natural language processing, Healthify supports data-driven insights into hospitalization trends, disparities, and geographic spread.
