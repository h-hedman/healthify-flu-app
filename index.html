Project Overview

This web app provides influenza-associated hospitalization data sourced from the FluSurv-NET FluView system (2009â€“2025).
This analysis is based on confirmed hospitalizations of seasonal influenza cases in children and adults.

FluSurv-NET conducts population-based surveillance for laboratory-confirmed influenza hospitalizations across multiple U.S. sites.
By leveraging this dataset, this tool analyzes trends in hospitalizations, demographic disparities, and disease burden.

This data-driven approach applies epidemiological and machine learning methods to enhance public health surveillance
and support informed decision-making on seasonal influenza mitigation strategies.
